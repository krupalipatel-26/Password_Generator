# Password Generator

A simple and secure Password Generator built using Python. This application generates random passwords based on user-defined length and ensures strong password security by combining uppercase letters, lowercase letters, numbers, and special characters.

## Features
- Generate random passwords
- User-defined password length
- Includes uppercase letters (A-Z)
- Includes lowercase letters (a-z)
- Includes numbers (0-9)
- Includes special characters (!,@,#,$,etc.)
- Strong and secure password generation

## Technologies Used
- Python
- Random Module
- String Module

## Learning Outcomes
- Python fundamentals
- Random number generation
- String manipulation
- User input handling
- Conditional statements
- Loops and lists
